# AI Email Summarization & Categorization Automation (Python)

This project is an automation tool that:
- Fetches emails from a Gmail inbox using IMAP  
- Converts each email into a clean `.txt` file  
- Generates short summaries of email content  
- Categorizes emails as **BUSINESS**, **PERSONAL**, **PROMOTION**, or **OTHER**  
- Detects priority level (**HIGH** / **NORMAL**)  
- Outputs everything into a structured CSV file  

Perfect for freelancers, small teams, and anyone who needs to scan many emails quickly.
